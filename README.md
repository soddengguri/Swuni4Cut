## 슈니네컷
학교에서 찍는 페이스타투 AR 인생네컷 (Face tattoo AR four Cuts taken at school.)


## 프로젝트 설명
- 학교 내, 사진이 잘 나오는 포토스팟에 '슈니네컷' 브로셔를 제공하여 AR 촬영 서비스 사용을 유도합니다.
- 학부 수업 프로젝트로 제작한 AR 카메라 어플입니다.

## 개요
- 프로젝트 이름 : 슈니네컷 (Swuni4Cut)
- 프로젝트 진행기간 : 2022.10-2022.12
